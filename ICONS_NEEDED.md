# Required Icon Assets for Report Rewrite Feature

The Report Rewrite feature requires two new icon files to be added to the `/public` directory:

## 1. rewrite.png
- **Location:** `/public/rewrite.png`
- **Size:** 16x16 or 32x32 pixels
- **Design suggestion:** Circular arrow or refresh symbol
- **Usage:** Toolbar button to open the Report Rewrite modal
- **Reference style:** Similar to existing icons like `/public/report.png`

## 2. undo.png
- **Location:** `/public/undo.png`
- **Size:** 16x16 or 32x32 pixels
- **Design suggestion:** Curved arrow pointing left
- **Usage:** Toolbar button to undo the last report rewrite
- **Reference style:** Similar to existing icons like `/public/new-conversation.png`

## Creating the Icons

You can create these icons using:
- **Image editor:** Photoshop, Figma, Sketch, etc.
- **Icon libraries:** [Heroicons](https://heroicons.com/), [Lucide](https://lucide.dev/), [Phosphor Icons](https://phosphoricons.com/)
- **Online tools:** [Favicon.io](https://favicon.io/), [Icon Generator](https://www.favicon-generator.org/)

## Icon Style Guidelines

To match the existing 3Panel icon style:
- Simple, minimal design
- Monochrome (gray/black)
- Clean lines
- 16x16 or 32x32 PNG format
- Transparent background

## Temporary Workaround

Until proper icons are created, the buttons will display with broken image icons. The feature will still be fully functional - only the visual icon will be missing.
